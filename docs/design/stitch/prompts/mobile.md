# Stitch Prompt — Mobile Responsive Pass

Create a mobile responsive pass for SERVIER Potion Lab.

**Platform:** Mobile web, 390px width baseline.

**Required behavior:**

1. Navigation collapses or wraps without horizontal overflow.
2. Dashboard hero remains compact; the primary CTA and launchpad are reachable without excessive scrolling.
3. Composer stacks cauldron protocol above ingredient selection; three sockets remain visible and touch-friendly.
4. Recipe cards and potion ledger become single-column, with guide buttons at least 44px tall.
5. Inventory ledger becomes a single-column stock list with decrement/recharge buttons grouped per ingredient.
6. No horizontal scrolling, no clipped cards, no sticky element hiding content.

Use the design system from `DESIGN.md`. Preserve all desktop route semantics and labels.
